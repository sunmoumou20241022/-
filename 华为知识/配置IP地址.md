# 配置IP地址

> **知识关联**：二层交换 → [[VLAN 跨交换机通信]] [[STP生成树]] | 三层安全 → [[ACL]] [[📝 eNSP 网络设备 AAA 认证实]] | 基础 → [[二进制]]

本次会议主要讲解了ESP的系统配置方法，并演示了如何使用网络仿真软件搭建拓扑、配置IP及进行基本的路由通信。

## **小结**

**1. ESP系统配置方法**

- 关闭Windows系统的“内核隔离”功能。
- 通过“组策略管理器”禁用“Driver Guard”中的“基于虚拟化的安全”选项。
- 完成上述步骤后重启电脑即可启用ESP。

**2. 网络仿真软件操作演示**

- **搭建拓扑**：演示了如何使用软件创建一个包含AR1、AR2、AR3三台路由器的拓扑结构，并为AR1和AR2添加SA（串口卡）。
- **连接设备**：讲解了如何使用以太网线和串口线将不同设备连接起来。
- **基础配置**：
    - 进入设备的用户视图（以“>”开头），输入“sys”进入系统视图（以“[”开头）。
    - 使用“sysname”命令修改设备名称以便区分。
    - 使用“interface”命令进入特定接口（如GE0/0/0），再使用“ip address”命令为接口配置IP地址和子网掩码。
    - 使用“dis ip interface brief”命令查看接口IP配置及状态。
- **串口接口配置**：演示了如何通过“interface se”命令进入串口（S0/0/0）接口，并为其配置IP地址。

**3. 网络通信验证**

- 配置完成后，演示了如何使用“ping”命令在不同设备间进行通信，验证网络连通性。
