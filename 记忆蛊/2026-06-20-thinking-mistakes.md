# 思维错误修正 — 2026-06-20

## 三个错误

### 错误1：迎合而不是质疑
用户说什么，我就在上面加码，没有反过来问"这真的靠谱吗？"
**修正**：先质疑，再认同。如果逻辑有问题，直接说出来。

### 错误2：用国内渠道查信息 / 用中文搜索
第一反应是百度/知乎/CSDN/闲鱼，搜索也用中文关键词，但国内AI信息严重滞后和堵塞。
而且用中文访问国外网站/文档本身就容易出问题。
**修正**：
- 所有搜索默认用英文关键词
- 访问国外网站/文档用英文
- 搜索问题也用英文
- 信息获取顺序：先官网，再GitHub等其他渠道
  - 官网可能下架/过期，信息反而别的地方有，所以每个渠道都要查
  - 但官网先查
- 每条信息必须标注：
  - 来源（URL/平台名）
  - 发布时间（日期）
  - 不标注来源 = 不可信
- 所有渠道都查完还是找不到 → 直接说"找不到"，不猜不编

### 错误3：给笼统建议而不是具体事实
"信息差赚钱"谁都知道，等于什么都没说。
**修正**：基于具体事实给建议，没有事实就说"我不知道"，不要编。
每条建议必须附上信息来源和时间，否则不说。

## 关于信息差的诚实判断

信息差分两种：
- **消失得快**：平台免费额度、安装教程、认证白嫖 → 3天内全网传遍，进去也是9.9的命运
- **消失得慢**：需要真正理解的事、需要持续更新的事、需要信任的事

但具体现在能赚什么 → 我不知道，因为没有最新国外信息。

## 下一步（等网络好了）

1. 扫 GitHub trending / Reddit / HackerNews / Product Hunt
2. 找最新的AI免费工具、新平台、新机会
3. 再看国内有没有人已经做了 → 判断信息差窗口
4. 基于事实给建议
