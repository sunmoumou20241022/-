# 换新电脑恢复笔记和记忆

## 1. 安装必要软件

- [Git](https://git-scm.com/downloads)
- [Obsidian](https://obsidian.md/download)
- [Claude Code](https://claude.ai/code)（可选，如果需要 AI 辅助）

## 2. 克隆 Vault（下载所有笔记）

```powershell
# 选一个你想要的目录，比如 D: 盘
cd D:\

# 克隆仓库
git clone https://github.com/sunmoumou20241022/-.git 网安

# 进入目录
cd 网安
```

> 仓库是私有的，首次克隆会要求登录 GitHub 或输入 Token。

## 3. 用 Obsidian 打开 Vault

1. 打开 Obsidian
2. 点击「打开文件夹作为仓库」
3. 选择刚才克隆的文件夹（如 `D:\网安`）
4. 完成，所有笔记都在了

## 4. 恢复 Claude 记忆（如果使用 Claude Code）

```powershell
# 创建 Claude 记忆目录（如果不存在）
New-Item -ItemType Directory -Path "$env:USERPROFILE\.claude\projects\C--Users-Administrator\memory" -Force

# 复制记忆文件
Copy-Item "D:\网安\.claude-memory\*.md" "$env:USERPROFILE\.claude\projects\C--Users-Administrator\memory\" -Force
```

复制完成后，Claude Code 就能读取到所有历史记忆了。

## 5. 日常同步笔记

每次写完笔记后，在 Vault 目录下运行：

```powershell
cd D:\网安
git add .
git commit -m "更新笔记"
git push
```

## 6. 换电脑后拉取最新笔记

如果另一台电脑上也克隆过这个仓库，只需要：

```powershell
cd D:\网安
git pull
```

就能同步到最新内容。
