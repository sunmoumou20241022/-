# 云端渗透跳板机搭建指南 (VPS + VLESS + Kali)

**核心目标**

**从零打造一台具备极速跨国网络、完美防封伪装、且能随时执行安全渗透测试的专属云端服务器。**

---

**底层环境**

**操作系统**：Debian Linux

**节点选择**：海外节点（如 Vultr 硅谷机房）

---

**第一阶段：服务器基础优化与安全加固**

**1. 开启 BBR 涡轮增压（极速网络）**

**操作流程**：

通过 SSH 连接云服务器。

终端输入 x-ui 进入管理菜单。

输入数字 24（开启 BBR）。

完成后输入 y 重启服务器。

**原理说明**：启用 Google 拥塞控制算法，大幅降低跨国传输丢包率，显著提升速度。

---

**2. 申请 SSL 证书（面板加密防窃听）**

**目的**：防止面板数据明文传输，消除浏览器红字警告，保障安全。

**操作步骤**：

**开端口**（确保证书机构可访问）：

iptables -I INPUT -p tcp --dport 80 -j ACCEPT

**申请证书**：

输入 x-ui 进入菜单。

选择设置 SSL 证书，选择选项 2（为纯 IP 申请证书）。

**访问方式**：后续登录面板需使用 [https://IP:](https://ip/)端口/后缀 格式。

---

**第二阶段：部署终极隐身节点 (VLESS + Reality)**

**采用 Reality 技术，防封伪装，将流量伪装成访问国际大厂官网。**

**1. 面板配置参数**

**协议 (Protocol)**：vless

**端口 (Port)**：443（全球标准 HTTPS 端口）

**安全性 (Security)**：reality

**目标网站 (Dest/SNI)**：

www.microsoft.com:443

www.amd.com:443

**密钥配置**：点击 Get New Cert 一键生成公钥和私钥

---

**2. 放行防火墙（关键）**

全新开启的 443 端口，需在服务器防火墙放行：

iptables -I INPUT -p tcp --dport 443 -j ACCEPT

iptables -I INPUT -p udp --dport 443 -j ACCEPT

---

**第三阶段：客户端终极形态（流量黑洞接管）**

**1. 电脑端（v2rayN + 混合战术核心）**

**节点导入**：复制面板节点链接，在 v2rayN 界面按 Ctrl+V 一键导入。

**全局接管（Tun 模式）**：打开底部“启用 Tun”开关，虚拟网卡强制接管所有软件流量（包括本地 VMware 虚拟机里的 Kali）。

**路由设置**：

日常使用：选择“绕过大陆(Whitelist)”

渗透测试或疑难杂症：选择“全局(Global)”

---

**2. 手机端（v2rayNG）**

**节点导入**：在电脑面板点击节点详情展示二维码，手机端点击 + 扫码一秒导入。

**智能分流**：进入设置 → 预定义规则 → 选择“绕过局域网及大陆地址”，实现国内抖音微信不卡，国外油管秒开。
